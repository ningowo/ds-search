# ds-search

## How to use

1. start the main method of RunCluster.java

2. start two data node with the following config

First:
-  server.port=8081
- search.server.nodeId=11
- search.server.port=6001
- search.server.host=localhost

Second:
- server.port=8080
- search.nodeId=10
- search.port=6000
- search.host=localhost
